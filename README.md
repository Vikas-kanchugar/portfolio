## ⚙️ Installation & Setup

Clone the repository

```bash
git clone https://github.com/Vikas-kanchugar/portfolio.git
```

Go to project directory

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
